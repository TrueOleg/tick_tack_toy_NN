# Tic-Tac-Toe-Machine-Leaning-Using-Tensor-Flow
Ticky is a Tensor Flow Model for playing Tic Tac Toe, which is trained to chose a move with best reward at any state of the board.

## Required
  <ol>
  <li>Python 3.5.2</li>
  <li>TensorFlow</li>
  <li>Flask</li>
  <li>numpy</li>
  </ol>

## How to Run
  python3 ./server.py<br>
